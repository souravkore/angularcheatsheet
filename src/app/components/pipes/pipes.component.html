Pipes in Angular

What is pipe in Angular ?

In Angular, a pipe is a simple function that takes an input value and transforms it into a desired output. It's often used to format data, such as converting dates, currencies, or filtering lists.

Why do we need pipes?
  • Data Formatting: Pipes make it easier to present data in a readable and consistent way. For example, you can use a date pipe to format a timestamp as a human-readable date.
  • Data Transformation: Pipes can transform data in various ways, like filtering arrays based on specific criteria or converting data types.
  • Reusability: Pipes can be reused throughout your Angular application, making your code more modular and maintainable.
Using Built-in Pipes

Transform data in the template using built-in pipes such as currency, date, uppercase, etc.

Example: {{ value | currency:'USD' }} converts a number to currency format.

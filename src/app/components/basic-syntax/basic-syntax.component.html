Angular Basics Cheatsheet

Creating a New Angular Project

To create a new Angular project, use the Angular CLI. Run the following command:

ng new project-name

Output: This command will create a new folder with Angular's default files and structure.

Compiling the Angular Project

To compile the Angular project, use the following command:

ng serve

Output: This command will compile the project and serve it on localhost:4200.

Interpolation

Interpolation allows you to embed the result of an expression inside the HTML using double curly braces {{curlybraces}}.

{{ message }}

Output: Hello, Angular!

Property Binding

Property binding allows you to set properties of HTML elements or directives using [property].

Output: The input field will be pre-filled with {{name}}.

Event Binding

Event binding allows you to listen to user events like clicks or keypresses using (event).

Output: When you click the button, an alert will be triggered with the message "Button Clicked!".

Two-Way Data Binding

Two-way data binding binds both the property and the event, allowing for synchronization between the model and the view using [(ngModel)].

Your name is: {{ name }}

Output: Any change in the input field will update the name variable, and the paragraph will reflect the updated name.

Angular Directives Cheatsheet

*ngIf Directive
Description

The '*ngIf' directive conditionally includes a template based on a boolean expression. If the expression evaluates to true, the template is included in the DOM, otherwise it's removed.

Syntax
{{code}}
Example

This text is conditionally visible based on the value of 'isVisible'.

*ngFor Directive
Description

The '*ngFor' directive is used to repeat a block of HTML for each item in a list. It iterates over the list and renders the element for each item.

Syntax
{{ngfor}}
Example
  • {{ item }}
[ngClass] Directive
Description

The '[ngClass]' directive allows you to dynamically add or remove CSS classes on an element based on conditions.

Syntax
{{ngclass}}
Example

This paragraph has conditional styling.

[ngStyle] Directive
Description

The '[ngStyle]' directive allows you to set multiple inline CSS styles for an element dynamically based on expressions.

Syntax
{{ngstyle}}
Example

This text color changes dynamically.
